前言

​ docker所在服务器，运行了一段时间后，发现服务器磁盘目录快不够用了。通过du -h --max-depth=1 / 逐级目录排查，发现/var/lib/docker目录文件过大。通过以下方法，解决该问题。

转移数据修改docker默认存储位置

有多种方式修改docker默认存储位置。
最好是在docker安装完后，第一时间修改docker默认存储位置为其他大目录或者磁盘中。规避迁移数据过程中造成的风险。
