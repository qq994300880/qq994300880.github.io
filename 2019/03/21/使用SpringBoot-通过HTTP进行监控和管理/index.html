通过HTTP进行监控和管理

如果您正在开发Web应用程序，则Spring Boot Actuator会自动配置所有已启用的端点以通过HTTP公开。默认约定是使用id前缀/actuator为URL路径的端点。例如，health暴露为/actuator/health。提示：Spring MVC，Spring WebFlux和Jersey本身支持Actuator。
